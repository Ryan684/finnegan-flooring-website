# Finnegan Flooring — Website Prototype

A modern single-page redesign prototype for Finnegan Flooring Limited.

## Live preview

This repository is configured for GitHub Pages. Once Pages is enabled
(Settings → Pages → Source: `main` / `(root)`), the site is served at:

`[https://ryan684.github.io.github.io/finnegan-flooring-website/](https://ryan684.github.io/finnegan-flooring-website/)`

## Local preview

Double-click `index.html` to open it in your browser. No build step required.

## Files

- `index.html` — the full single-page site (HTML, CSS, JS all inline)
- `tweaks-panel.jsx` — design-tweak controls (only active inside the
  design editor; on the public site it loads but stays hidden, which is intended)

## Editing

Significant changes are done in the design editor where this prototype
lives. To publish a new version, replace `index.html` in this repo —
GitHub Pages will redeploy within a minute.
